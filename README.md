# My Lifts Mall — Store Owners Portal

A website for a Roblox mall: verified Discord members browse stores, vote in
elections for vacant stores, and store owners upload `.rbxl` files for the game
owner to review and publish. Notifications go out as Discord DMs via the
companion bot (`mylebot`).

React + Mantine · Fastify + Prisma + PostgreSQL · Docker.

```powershell
docker compose up --build -d     # http://localhost:8080
```

Technical details for AI coding agents live in [CLAUDE.md](CLAUDE.md).
